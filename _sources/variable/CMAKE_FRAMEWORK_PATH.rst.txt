CMAKE_FRAMEWORK_PATH
--------------------

指定\ :command:`find_library`、\ :command:`find_package`、\ :command:`find_path`\
和\ :command:`find_file`\ 命令使用的macOS框架的搜索路径的目录\
:ref:`分号分隔列表 <CMake Language Lists>`。

There is also an environment variable :envvar:`CMAKE_FRAMEWORK_PATH`, which is used
as an additional list of search directories.
