CMAKE_EDIT_COMMAND
------------------

:manual:`cmake-gui(1)`\ 或\ :manual:`ccmake(1)`\ 的全路径。仅在不使用任何\
:ref:`Extra Generators`\ 时为\ :ref:`Makefile Generators`\ 和\
:ref:`Ninja Generators`\ 定义。

This is the full path to the CMake executable that can graphically
edit the cache.  For example, :manual:`cmake-gui(1)` or :manual:`ccmake(1)`.
