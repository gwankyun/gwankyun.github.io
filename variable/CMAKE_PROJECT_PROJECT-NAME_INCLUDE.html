CMAKE_PROJECT_<PROJECT-NAME>_INCLUDE¶

一个CMake语言文件，作为指定<PROJECT-NAME>作为项目名的project()命令调用的最后一步。这是为了在不修改源代码的情况下将自定义代码注入到项目构建中。有关在project()调用期间可能包含的文件的更详细讨论，请参阅代码注入。

在 3.29 版本加入: This variable can be a semicolon-separated list of CMake language files to be included sequentially. It can also now refer to module names to be found in CMAKE_MODULE_PATH or as a builtin CMake module.

See also the CMAKE_PROJECT_<PROJECT-NAME>_INCLUDE_BEFORE, CMAKE_PROJECT_INCLUDE, CMAKE_PROJECT_INCLUDE_BEFORE, and CMAKE_PROJECT_TOP_LEVEL_INCLUDES variables.
